#Projeto da aula aberta - Comunicação PC  - Microcontrolador / Arduino utilizando Firmata (Interface Gráfica com TKInter)

Nesta nova vídeo aula aberta vamos utilizar os conceitos de comunicação serial que estamos estudando para criar aplicações com comunicação entre computadores e microcontroladores (ou Arduino), para isso utilizaremos Python, TK Inter para criação de interface gráfica (GUI) e Firmata par comunicação!
