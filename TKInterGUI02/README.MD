#Projeto da aula aberta - Criação de Iterface Gráfica com Python e Tkinter (Parte 2)

Nesta nova vídeo aula aberta vamos dar continuidade no aprendizado de alguns conceitos fundamentais sobre criação de interfaces gráficas utilizando Python e a biblioteca gráfica nativa Tkinter e
dessa forma vamos criar uma aplicação gráfica com botões, label, canva (criando uma sinalização gráfica).
